# Лендинг для wellness-продуктов

## Обзор проекта
Современный многостраничный лендинг с природной цветовой палитрой и плавными анимациями. Цель лендинга — мотивировать посетителей перейти на основной сайт. Проект создан на React + Vite + TypeScript + Tailwind CSS.

## Последние изменения (26 ноября 2025)

### Полная реструктуризация продуктов и секций
- 6 разных товаров (не все пластыри):
  1. **Лечебный пластырь** - для точечного воздействия
  2. **Очиститель воды** - структурирует и очищает воду
  3. **Очелье** - налобная повязка для головы
  4. **Наочники** - для глаз и зрения
  5. **Кушак** - пояс для спины и живота
  6. **Тесьма** - для рук, ног и суставов

### Структура каждого варианта
Все 6 вариантов дизайна теперь содержат полный набор секций:
- Hero (главный баннер)
- О продукции (краткое описание)
- Каталог продуктов (6 товаров с иконками)
- Как это работает (3 шага)
- Отзывы клиентов (4 отзыва)
- FAQ (4 вопроса)
- CTA (призыв перейти на сайт, не форма заказа)

### 6 вариантов дизайна
1. **Утренняя роса** - свежие мятные тона, капельки росы, лёгкий и воздушный
2. **Лесная тропа** - тёплые земляные тона, древесные текстуры
3. **Цветущий луг** - нежные розовые тона, весеннее настроение
4. **Травяная аптека** - винтажный ботанический стиль, мудрость веков
5. **Горный источник** - прохладные голубые тона, горные силуэты
6. **Солнечный луг** - тёплые золотистые тона, солнечные лучи

### Изменения
- Убран блок ингредиентов (по запросу)
- CTA кнопки: "Перейти на сайт", "Перейти в каталог" с иконкой внешней ссылки

## Архитектура проекта

### Структура файлов
```
/
├── src/
│   ├── pages/
│   │   ├── DesignVariants.tsx  # 6 вариантов дизайна (главная)
│   │   └── Home.tsx            # Альтернативная страница
│   ├── App.tsx                 # Роутинг приложения
│   ├── main.tsx                # Точка входа React
│   └── index.css               # Глобальные стили
├── index.html                  # HTML шаблон
├── vite.config.ts              # Конфигурация Vite
├── package.json                # Зависимости проекта
└── tsconfig.json               # Настройки TypeScript
```

### Технологический стек
- **React 19** - UI библиотека
- **TypeScript** - типобезопасность
- **Vite** - быстрый сборщик и dev сервер
- **Tailwind CSS v4** - utility-first CSS фреймворк
- **Framer Motion** - анимации и transitions
- **Wouter** - лёгкий роутинг
- **Lucide React** - иконки

### Конфигурация
- **Dev сервер**: порт 5000, host 0.0.0.0
- **Production build**: статический сайт в папке `dist/`
- **HMR**: настроен для работы через WebSocket Secure (wss://)

## Команды
- `npm run dev` - запуск dev сервера (порт 5000)
- `npm run build` - сборка для production
- `npm run preview` - предпросмотр production сборки
