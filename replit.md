# Лендинг для лечебного пластыря

## Обзор проекта
Современный многостраничный лендинг с природной цветовой палитрой и плавными анимациями. Проект создан на React + Vite + TypeScript + Tailwind CSS.

## Последние изменения (26 ноября 2025)

### 6 вариантов дизайна
Созданы 6 уникальных вариантов дизайна с природной тематикой:

1. **Утренняя роса** - свежие мятные тона, капельки росы, лёгкий и воздушный
2. **Лесная тропа** - тёплые земляные тона, древесные текстуры, годовые кольца дерева  
3. **Японский сад** - минималистичный дзен-стиль, японские иероглифы, спокойствие
4. **Травяная аптека** - винтажный ботанический стиль, декоративные рамки
5. **Горный источник** - прохладные голубые тона, горные силуэты, чистота
6. **Солнечный луг** - тёплые золотистые тона, солнечные лучи, уют

Переключение между вариантами через стрелки в верхней панели.

## Архитектура проекта

### Структура файлов
```
/
├── src/
│   ├── pages/
│   │   └── Home.tsx          # Главная страница лендинга
│   ├── App.tsx               # Роутинг приложения
│   ├── main.tsx              # Точка входа React
│   └── index.css             # Глобальные стили
├── index.html                # HTML шаблон
├── vite.config.ts            # Конфигурация Vite
├── package.json              # Зависимости проекта
└── tsconfig.json             # Настройки TypeScript
```

### Технологический стек
- **React 19** - UI библиотека
- **TypeScript** - типобезопасность
- **Vite** - быстрый сборщик и dev сервер
- **Tailwind CSS v4** - utility-first CSS фреймворк
- **Framer Motion** - анимации и transitions
- **Wouter** - лёгкий роутинг
- **Lucide React** - иконки

### Конфигурация
- **Dev сервер**: порт 5000, host 0.0.0.0
- **Production build**: статический сайт в папке `dist/`
- **HMR**: настроен для работы через WebSocket Secure (wss://)

## Содержание лендинга
1. **Hero секция** - чистый дизайн с центрированным контентом (без летающих листиков)
2. **Механизм действия** - 4 этапа с адаптивными карточками
3. **Линейка продуктов** - 4 вида пластырей с иконками
4. **Timeline преимуществ** - упрощенный card grid дизайн
5. **Интерактивные табы** - преимущества с переключением
6. **Натуральные ингредиенты** - список компонентов
7. **FAQ секция** - часто задаваемые вопросы
8. **Отзывы клиентов** - 3 отзыва с рейтингами
9. **Финальный CTA** - призыв к действию с гарантиями

## Команды
- `npm run dev` - запуск dev сервера (порт 5000)
- `npm run build` - сборка для production
- `npm run preview` - предпросмотр production сборки
